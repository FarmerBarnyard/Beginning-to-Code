Cascading Selectors

When using multiple selectors in html, and stylising it in css, the rule that is the most specific in css will be the one that shows up when the website is published.

An additional rule, is that ID selectors will always take superiority over class selectors due to the specificity of an ID selector.

If specificity is the same across the board; class, ID, descendant and child selectors, then the last defined rule will be the one presented in the webpage.
