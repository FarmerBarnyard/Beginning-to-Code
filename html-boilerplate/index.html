Stylising Text Types

To separate paragraphs the "p" function must be used

When you want to type something in bold, the strong function must be used

To italicise something, the em function must be used

For extra headings, h1-h6 can be used for different levels of headings

Creating Lists

To create an unordered list, for groceries or similar type lists, use the ul function.

For example:

  • Item 1
  • Item 2
  • Item 3

If you wanted an ordered list, however, like a series of tasks, use the ol function.

For example:

  1. Item 1
  2. Item 2
  3. Item 3

Creating Links

To hyperlink to another webpage or interior page, use the "a" command. For example; if we wanted to visit the Odin Project...

Click Here for The Odin Project

Displaying Images

To place an image into a webpage it is a simple as using the "img" command
